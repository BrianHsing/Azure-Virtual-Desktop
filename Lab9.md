## Lab9.開始使用 WVD
 - 使用 Web 用戶端連線，任何支援 HTML5 的瀏覽器應該皆可正常運作。<br>
	- 開啟任何一台桌上型、筆記型電腦，開啟瀏覽器，並輸入 https://rdweb.wvd.microsoft.com/arm/webclient<br>
	- 登入 Azure AD 使用者帳號密碼，此步驟為第一階段驗證，與 Azure AD 驗證，您可以在這個步驟使用 MFA、條件式存取。<br>
